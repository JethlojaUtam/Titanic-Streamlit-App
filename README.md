# Titanic-Streamlit-App
Titanic Survival Prediction Web App built using Streamlit and Machine Learning. This app allows users to explore data insights, perform EDA, and predict passenger survival probability interactively.

# 🚢 Titanic Survival Prediction App

This is a Machine Learning web application built using **Streamlit** that predicts whether a passenger survived the Titanic disaster.

---

## 📌 Project Overview

The goal of this project is to build a classification model that predicts passenger survival based on features like age, gender, class, fare, and more.

---

## ⚙️ Features

- 📊 Interactive Dashboard  
- 🎯 Survival Prediction System  
- 📈 Exploratory Data Analysis (EDA)  
- 🧠 Machine Learning Model (KNN)  
- ⚡ Real-time Predictions  

---

## 🛠 Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit  
- Matplotlib  
- Seaborn  

---

## 🤖 Machine Learning Model

- Algorithm: **K-Nearest Neighbors (KNN)**
- Accuracy: **82%**

---

## 📊 Dataset

- Titanic dataset with passenger details such as:
  - Age  
  - Gender  
  - Passenger Class  
  - Fare  
  - Embarked Port  

---
