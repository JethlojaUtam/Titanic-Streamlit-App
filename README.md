## 📌 Project Overview

The objective of this project is to build a **classification model** that predicts passenger survival using features like age, gender, class, fare, and embarkation point.

This project demonstrates a complete **end-to-end ML pipeline**:
- Data preprocessing  
- Feature engineering  
- Model training  
- Deployment using Streamlit  

---

## ⚙️ Features

- 📊 Interactive Dashboard  
- 🎯 Real-time Survival Prediction  
- 📈 Exploratory Data Analysis (EDA)  
- 🧠 Machine Learning Model  
- ⚡ User-friendly Interface  

---

## 🧠 Machine Learning Model

- Algorithm: **Random Forest Classifier**
- Accuracy: **85%+**
- Improvements:
  - Feature Engineering (FamilySize, IsAlone)
  - Hyperparameter tuning

---

## 📊 Dataset

- Titanic dataset including:
  - Age  
  - Gender  
  - Passenger Class  
  - Fare  
  - Embarked Port  

---

## 🔄 Data Preprocessing

- Handling missing values  
- Label Encoding  
- Feature Engineering:
  - `FamilySize`
  - `IsAlone`
